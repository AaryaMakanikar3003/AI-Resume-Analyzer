# AI-Resume-Analyzer
An AI-powered resume analyzer that extracts key information from uploaded resumes, evaluates candidate skills, and provides structured insights using NLP and LLMs. Built with Streamlit, LangChain, and vector embeddings for fast, accurate, and automated resume assessment.
